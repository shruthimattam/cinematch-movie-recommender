\# 🎬 CineMatch — Movie Recommendation System



An AI-powered content-based movie recommendation system built with Python and Machine Learning.



\## 🚀 Live Demo

Type any movie name and get 5 similar recommendations instantly!



\## 🧠 How It Works

1\. Movie genres are converted into numerical vectors using \*\*TF-IDF Vectorization\*\*

2\. \*\*Cosine Similarity\*\* calculates the angle between vectors to find similar movies

3\. Top 5 most similar movies are returned ranked by similarity score



\## 🛠️ Tech Stack

\- \*\*Python\*\* — Core language

\- \*\*Pandas\*\* — Data manipulation

\- \*\*Scikit-learn\*\* — TF-IDF \& Cosine Similarity

\- \*\*Streamlit\*\* — Web application UI
